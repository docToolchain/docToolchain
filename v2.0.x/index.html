docToolchain

Create awesome Docs the easy way!

docToolchain is a collection of scripts which help you to create and maintain awesome technical documentation.

Take a look at all these great features and give our tutorial a try!

Based on best of breed open source technologies

We create the best documentation toolchain, so you don't have to.

AsciiDoc / Asciidoctor

We believe that AsciiDoc is the right choice for technical documentation. It has everything you need already available, so you don't have to choose which flavour to use.

Static-Site-Generator

There are hundreds of static site generators out there. We use jBake under the hood, but it shouldn't matter for you - just concentrate on your docs.

Open Source

Did we already mention that docToolchain is fully open source? No vendor lock in. You are free to modify docToolchain to fit your needs and one of our architectural goals is to help you in doing so.

Multi-Repository-Builds

Just kidding. Since all your dev environments will already contain a git client, we didn't add another one. Just use Git to create your Multi-Repository-Builds as explained in our tutorial.
